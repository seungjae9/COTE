###### 16506번 CPU
```?
		진수 처리하는방법
		ex) 숫자 5를 2진수로 바꾸는법
		
			tmp = format(5, 'b')
			파이썬에서 이런식으로 바꾸면 0b~~~~ 이런식으로 나오는게 아니라
			정확하게 101 같은 숫자만 나온다
			
			tmp = bin(5)
			이렇게해서 2진수로 바꾸게 되면 0b101 형태로 나옴
			
			'b' = bin = 2진수
			'o' = oct = 8진수
			'h' = hex = 16진수
```
